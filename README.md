# 一个简单的前后端分离的管理系统

##前端技术

| 技术 | 版本 | 官网 |
| :-----| ----: | :----: |
| Vue | 2.6.11 | https://vuejs.org/ |
| Vue-router | 3.2.0 |  	https://router.vuejs.org/ |
| Element | 2.15.6 | https://element.eleme.io |
| Axios | 0.26.1 | https://github.com/axios/axios |
| core-js | 3.6.5 | https://opencollective.com/core-js |

vue
